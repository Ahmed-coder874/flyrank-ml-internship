Content Action Playbook

Abstract

This project studies whether historical content performance signals can help prioritize pages for review.

Research Question

Can historical content performance signals be used to prioritize pages for content review?

Methodology

A baseline scoring method and Random Forest model were evaluated using historical content signals.

Results

Baseline accuracy: 54.5%
Random Forest accuracy: 69.2%
Time-aware validation: 59.0%

Limitations

The output is decision support only. Human review is required.
